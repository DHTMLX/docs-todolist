# unselect-item

### Description

@short: fires when unselecting an item

### Usage

```js
"unselect-item": ({
    id: string | number,
}) => void;
```
### Parameters

The callback of the **unselect-item** event can take an object with the following parameters:

- `id` - (required) the id of the item
