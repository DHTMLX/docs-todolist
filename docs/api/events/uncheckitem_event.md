# uncheck-item

### Description

@short: fires when the item is unchecked

### Usage

~~~js
"uncheck-item": ({
    id: string | number,
}) => void;
~~~

### Parameters

The callback of the **uncheck-item** event can take an object with the following parameters:

- `id` - (required) the id of the item
