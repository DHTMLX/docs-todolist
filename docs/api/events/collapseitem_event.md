# collapse-item

### Description

@short: fires when collapsing an item

### Usage

~~~js
"collapse-item": ({
    id: string | number,
}) => void;
~~~

### Parameters

The callback of the **collapse-item** event can take an object with the following parameters:

- `id` - (required) the id of the item
