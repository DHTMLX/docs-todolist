# assignUser()

### Description

@short: assigns a user to the specified task

### Usage

~~~js
assignUser({
    id: string | number,
    userId: string | number,
}): void;
~~~

### Parameters

- `id` - (required) the id of the task
- `userId` - (required) the id of the user to be assigned to the task
