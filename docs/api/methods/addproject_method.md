# addProject()

### Description

@short: Adds a new project

### Usage

~~~js
addProject({
    id?: string | number,
    project?: object
}): void;
~~~

### Parameters

- `id` - (optional) the ID of a new project
- `project` - (optional) an object with parameters of the project:
  - `label` - (optional) the name of the project
