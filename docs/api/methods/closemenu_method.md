# closeMenu()

### Description

@short: Closes the opened menu of a task/project

### Usage

~~~js
closeMenu({
    id?: string | number
}): void;
~~~

### Parameters

- `id` - (optional) the ID of a task/project the menu of which must be closed. If not specified, the method will close the currently opened menu
