# unassignUser()

### Description

@short: Unassigns a user from the specified task

### Usage

~~~js
unassignUser({
    id: string | number,
    userId: string | number
}): void;
~~~

### Parameters

- `id` - (required) the id of a task
- `userId` - (required) the id of the user which to be unassigned from the task
