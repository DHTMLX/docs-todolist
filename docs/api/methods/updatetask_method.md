# updateTask()

### Description

@short: updates properties of a task

### Usage

~~~js
updateTask({
    id: string | number;
    task: object;
}): void;
~~~

### Parameters

- `id` - (required) the id of the task to be updated
- `task` - (required) an object with a hash of task's properties which need to be updated
