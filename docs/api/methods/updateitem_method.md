# updateItem()

### Description

@short: updates properties of the item

### Usage

~~~js
updateItem({
    id: string | number;
    item: object;
}): void;
~~~

### Parameters

- `id` - (required) the id of the item to be updated
- `item` - (required) an object with a hash of item's properties which need to be updated
