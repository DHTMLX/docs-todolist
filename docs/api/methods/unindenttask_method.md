# unindentTask()

### Description

@short: Promotes the nesting level of the task to one higher level, depending on the nearby task in the data set

### Usage

~~~js
unindentTask({
    id: string | number,
}): void;
~~~

### Parameters

- `id` - (required) the id of a task
