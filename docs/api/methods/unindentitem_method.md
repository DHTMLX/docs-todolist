# unindentItem()

### Description

@short: Promotes the nesting level of the item to one higher level, depending on the nearby item in the data set

### Usage

~~~js
unindentItem({
    id: string | number,
}): void;
~~~

### Parameters

- `id` - (required) the id of the item
