# closeTaskEditor()

### Description

@short: Closes the inline editor of a task

### Usage

~~~js
closeTaskEditor({
    id: string | number,
    save?: boolean,
}): void;
~~~

### Parameters

- `id` - (required) the id of a task
- `save` - (optional) defines whether the changes should be saved after closing the editor; **true** by default
