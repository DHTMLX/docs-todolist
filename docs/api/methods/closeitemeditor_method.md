# closeItemEditor()

### Description

@short: closes the inline editor of an item

### Usage

~~~js
closeItemEditor({
    id: string | number,
    save?: boolean,
}): void;
~~~

### Parameters

- `id` - (required) the id of the item
- `save` - (optional) defines whether the changes should be saved after closing the editor; **true** by default
