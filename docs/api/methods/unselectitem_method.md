# unselectItem()

### Description

@short: unselects an item(s)

### Usage

~~~js
unselectItem({
    id: string | number,
}): void;
~~~

### Parameters

- `id` - (optional) the id of the item to be unselected

:::info
To unselect all previously selected items, call the **unselectItem()** method without parameters
:::
