# unselectTask()

### Description

@short: unselects a task(s)

### Usage

~~~js
unselectTask({
    id: string | number,
}): void;
~~~

### Parameters

- `id` - (optional) the id of the task to be unselected

:::info
To unselect all previously selected tasks, call the **unselectTask()** method without parameters
:::
